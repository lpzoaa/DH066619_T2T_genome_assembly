# Onion Genome Assembly Pipeline

This repository contains scripts and resources used for generating the DH066619-T2T onion genome assembly and performing population genomic analyses.

## Features
- PacBio HiFi & ONT-UL data processing
- Hybrid assembly using Hifiasm
- Gap filling workflow for T2T construction
- Variant calling pipeline (bcftools)
- Population structure analysis (ADMIXTURE, PCA, IBS)
- Repeat annotation (LTR, TRIM, LARD)
